# Linked Lists in C

This is a simple exercise to create linked list data structures useing the C programming language.

## Environment

- Kubuntu 18.10 - GNU/Linux
- Intel CPU - 64 bits
- GCC Version 7.3.0
- CMake 3.13.0
- NVim Version 0.2.2
